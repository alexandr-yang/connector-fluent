# Конструктор HTTP-сессии

### Сессия

Создать новую HTTP-сессию

**Возвращаемое значение:**
- ```ОбработкаОбъект.КонструкторHTTPСессии```

---

**Параметры:**
- Ключ - ```Строка```
- Значение - ```Строка```

- ```ОбработкаОбъект.КонструкторHTTPСессии```
